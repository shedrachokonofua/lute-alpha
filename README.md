# TypeScript + Node.js Starter

Personal TypeScript + Node.js project scaffold.

## Overview

- Typescript 4.1
- Purify
- Jest + Jest Extended
- Prettier
- ESLint + Curated Extensions:
  - @typescript-eslint
  - eslint-plugin-unicorn
  - eslint-plugin-jest
  - eslint-plugin-jest-formatting
  - eslint-plugin-import
  - eslint-config-prettier

## Commands

### Build

```
npm run build
```

### Lint

```
npm run lint
```

Lint + Fix

```
npm run lint:fix
```

Lint + Fix + Watch

```
npm run lint:watch
```

### Run

```
npm start
```

Dev mode

```
npm run dev
```

### Test

```
npm test
```
